# 홈 화면 정의서

## 1. 화면 개요
- 화면명: 홈 화면
- 파일명: `home.html`
- 목적: 설문조사 제목 입력, 분석용 파일 업로드, 저장된 대시보드 조회, 분석 시작
- 기본 폭 기준: `900px`

## 2. 주요 사용자 목표
- 설문조사 제목을 입력한다.
- 코드북, 응답 데이터셋(숫자형), 응답 데이터셋(라벨형)을 업로드한다.
- 저장된 대시보드 리스트를 확인하거나 기존 대시보드로 진입한다.
- 유효성 검증을 통과하면 분석을 시작한다.

## 3. 화면 구성

### 3.1 상단 유틸 영역
- 좌측: `purple6studio` 로고
- 우측: `저장된 대시보드 리스트` 버튼
- 특징
  - 상단 유틸 영역과 본문 컨테이너는 동일한 가로 기준선 위에 정렬된다.
  - 리스트 버튼은 보조 액션으로, 메인 제목보다 존재감이 낮다.
  - 리스트 버튼 라운드 값은 일반 버튼과 동일한 `radius-sm`을 사용한다.

### 3.2 헤더 영역
- 중앙 정렬 제목: `설문조사 분석 대시보드 만들기`
- 메인 목적을 가장 먼저 인지하게 하는 1차 정보 영역

### 3.3 섹션 1: 설문조사 제목
- 타이틀: `1. 설문조사 제목을 입력해주세요.`
- 입력 필드
  - placeholder: `예: 2025 직장인 사무환경 조사`
  - 최대 입력 길이: `50자`
- 보조 문구
  - `최대 50자까지 입력할 수 있습니다.`
- 오류 문구
  - `설문조사 제목을 입력해 주세요.`

### 3.4 섹션 2: 파일 업로드
- 타이틀: `2. 문항 코드북과 응답 데이터셋을 업로드해주세요.`
- 3열 업로드 카드
  - `문항 코드북`
  - `응답 데이터셋_숫자형`
  - `응답 데이터셋_라벨형`
- 각 카드 공통 구성
  - 기본 안내 문구
  - 허용 형식: `.csv`, `.xlsx`
  - `파일 선택` 버튼
  - 드래그 앤 드롭 지원
  - 업로드 성공 시 파일명 + `업로드 완료!`
  - `다시 선택하기` 버튼 제공

### 3.5 메인 액션
- 버튼명: `분석 시작`
- 활성 조건
  - 제목 입력 완료
  - 코드북 업로드 완료
  - 숫자형 응답 데이터 업로드 완료
  - 라벨형 응답 데이터 업로드 완료

### 3.6 가이드 배너
- 문구
  - 문항 코드북과 응답 데이터셋이 무엇인지 모르겠거나, 아직 만들지 않았다면 아래 가이드를 참고해 주세요.
- 링크 텍스트
  - `가이드 보러 가기 →`
- 현재 상태
  - 실제 링크는 미연결
  - placeholder alert 동작

### 3.7 저장된 대시보드 모달
- 모달 제목: `저장된 대시보드 리스트`
- 기능
  - 저장된 설문 목록 표시
  - 제목 클릭 시 해당 대시보드 진입
  - `삭제` 버튼으로 개별 삭제
  - 저장 건수는 상단 리스트 버튼 뱃지에 표시

## 4. 데이터 및 검증 규칙

### 4.1 제목 입력
- `maxlength="50"` 적용
- 공백만 입력된 경우 비활성 처리

### 4.2 코드북 검증
- 필수 컬럼
  - `question_no`
  - `question_label`
  - `response_type`
  - `data_column_role`

### 4.3 응답 데이터 공통 검증
- 필수 컬럼
  - `survey_year`
  - `respondent_no`

### 4.4 숫자형/라벨형 판별
- 숫자형
  - 응답 값이 숫자 코드 중심
- 라벨형
  - 응답 값이 라벨 텍스트 중심
- 잘못 업로드된 경우 해당 업로드 박스에 오류 메시지 표시

## 5. 저장 방식
- 저장 키: `localStorage['p6s.surveys']`
- 현재 선택 설문
  - `sessionStorage['survey.currentId']`
  - `sessionStorage['survey.title']`
- 저장 항목
  - 설문 제목
  - 생성 시각
  - 업로드 파일명
  - 업로드 파일 내용

## 6. 현재 UX 특징
- 첫 화면에서 주요 요소가 한눈에 보이도록 전체 세로 밀도 조정 완료
- 로고와 리스트 버튼은 메인 제목보다 보조적으로 보이도록 위계 정리 완료
- 리스트 버튼과 본문은 동일한 가로 기준선 위에 정렬 완료
- 로고와 제목, 가이드 배너와 푸터 간 간격 조정 완료
- 전체 폭은 `900px` 기준으로 정리됨

## 7. 제외 범위
- 가이드 링크 실제 연결
- 업로드 샘플 다운로드
- 저장된 대시보드 검색/정렬 기능
